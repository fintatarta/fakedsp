Files/Open.Source [ Functions ]

SOURCE

   function Open (Filename : String;
                  Format   : File_Type := Unknown)
                  return Data_Source_Access;

DESCRIPTION

Open a Data_Source based on the file with the specified filename and format (WAV, Text-based, ...). If the special filename "-" is used, the standard input is used.

If Format = Unknown, then:

  • if Filename = "-", format Text_File is used by default, otherwise
  • the format is guessed on the basis of the extension (maybe in the future we will check the content too).

Sampling frequency and number of channels are Read from the specified file or from the options Read from the filename
