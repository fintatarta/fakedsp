Fakedsp.card/State_Type [ Types ]

DESCRIPTION

The "virtual acquisition card" can be in one of 3 states: Sleeping, Running, End_Of_Data.

At the beginning, before calling the function Start, it is in Sleeping; after Start it goes Running until it ends the data from the source; when out of data, the card goes in End_Of_Data.
