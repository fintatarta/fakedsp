DESCRIPTION

The library Fakedsp allows to Write code with a DSP-style on a normal PC. I wrote this to help my students in doing their lab activities.

The user of the library will be most probably interested in the package Fakedsp.card that provides the interface to the "virtual DSP card."

Another package of interest is probably Fakedsp.Data_Streams.Files that implement the Data_Source/Data_Destination interfaces (defined in Fakedsp.Data_Streams) that represent the main abstraction for data I/O.
