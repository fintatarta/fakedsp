Data_Streams/Data_Destination [ Abstract Interfaces ]

SOURCE

   type Data_Destination is limited interface;


   type Data_destination_Access is access all Data_Destination'Class;

DESCRIPTION

Abstract interface representing the generic data destination. A concrete implementation of this interface needs to define:

  • procedure Write to output the next sample
  • function Max_Channel returning the number of the last channel
  • procedure Close that... Close the destination.
