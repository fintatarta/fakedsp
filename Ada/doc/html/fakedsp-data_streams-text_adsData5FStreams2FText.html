Data_Streams/Text [ Process ]

DESCRIPTION

This package provides a concrete implementation of interfaces Data_Source and Data_Destination that do I/O in Text format. Although not very efficient on the space occupation side, it is convenient for backup etc.

OPTIONS

It accepts option "fmt" that if equal to "float" force I/O in floating point format, otherwise signed 16 bit integers are used.

For example,

-::fmt=float

do I/O with floating point numbers to standard input/standard output
