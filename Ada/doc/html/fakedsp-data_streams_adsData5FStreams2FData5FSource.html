Data_Streams/Data_Source [ Abstract Interfaces ]

SOURCE

   type Data_Source is limited interface;

   type Data_Source_Access is access all Data_Source'Class;

DESCRIPTION

Abstract interface representing the generic data source. A concrete implementation of this interface needs to define:

  • procedure Read to get the next sample
  • function Sampling_Frequency that returns the sampling frequency of the source
  • function Max_Channel returning the number of the last channel
  • procedure Close that... Close the source.
