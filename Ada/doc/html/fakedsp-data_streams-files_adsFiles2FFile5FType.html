Files/File_Type [ Types ]

SOURCE

   type File_Type is (Wav_File, Text_File, Unknown);

DESCRIPTION

Enumeration type representing the currently recognized format. This type is here since we want to allow the user to specify the format when the source/destination is open, without any guessing on the library side.

It is a good idea to extend this type when a new format is added.
